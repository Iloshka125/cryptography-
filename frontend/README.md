# Frontend Auth Mock

Минимальное React-приложение на Vite с двумя страницами:

- `LoginPage` — вход по почте/телефону и паролю
- `RegisterPage` — регистрация с никнеймом, контактами и подтверждением пароля

Валидация работает на стороне клиента; вместо реальных запросов данные выводятся в консоль. Подключи свой API внутри обработчиков `handleSubmit`.

## Быстрый старт

```bash
cd frontend
npm install
npm run dev
```

### Сборка

```bash
npm run build
```
